Interactsh es una herramienta de código abierto para detectar interacciones fuera de banda. Está diseñada para detectar vulnerabilidades que causan interacciones externas.

**Enlace**: [Interactsh](https://app.interactsh.com/#/).  
**GitHub**: `https://github.com/projectdiscovery/interactsh.git`.


## Caracteristicas

- Interacción DNS/HTTP(S)/SMTP(S)/LDAP.
- CLI / Web / Burp / ZAP / Cliente Docker.
- Cifrado AES sin registro.
- TLS comodín basado en ACME automático con renovación automática.
- Entradas DNS para el servicio de metadatos en la nube.
- Control de respuesta HTTP dinámica.
- Servidor interactivo autohospedado.
- Soporte para múltiples dominios (autohospedado).
- Escucha NTLM/SMB/FTP(S)/RESPONDER (autohospedada).
- Comodines / Interacciones protegidas (autohospedadas).
- Índice personalizable / Alojamiento de archivos (autohospedado).
- Longitud de carga útil personalizable (autohospedado).
- Certificado SSL personalizado (autohospedado).
