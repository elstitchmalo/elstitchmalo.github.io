Burp Collaborator es una herramienta de pago de PortSwigger, que sirve para detectar interacciones fuera de banda. Está diseñada para detectar vulnerabilidades que causan interacciones externas.

**Enlace**: [Burp Collaborator](https://portswigger.net/burp/documentation/collaborator/uses#blind-sql-injection)

## Caracteristicas

A continuación se describen algunos casos de uso comunes:

- Interacción con servicios externos.
- Carga de recursos fuera de banda.
- Inyección SQL ciega.
- Scripting entre sitios a ciegas.
